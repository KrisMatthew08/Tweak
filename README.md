# Tweak: AI-Powered Instant Presentation Generator

*Precision. Design. Transformation.*

This is the **DIY presentation generator**, a feature of **Tweak** — a design-support service helping students get fast, professional academic visual materials.

Describe a topic, pick a template, and get back a ready-to-edit PowerPoint (`.pptx`) deck — generated by an LLM and assembled with `python-pptx`.

> Built on top of [SlideDeck AI](https://github.com/barun-saha/slide-deck-ai), an open-source AI presentation generator. The core slide-generation engine, prompts, and PPTX templates come from that project (under `src/slidedeckai`); this repo wraps it in a FastAPI backend and a small web UI for Tweak.


## How It Works

1. **AI Content Generation** — Given a topic, an LLM generates structured JSON content for the deck (title, slide headings, bullet points, key messages, icon/image keywords).
2. **Visual Enhancement** — Keywords from the JSON are used to fetch relevant images (via Pexels) and match icons for some slides.
3. **PPTX Assembly** — `python-pptx` builds the slide deck from the JSON using one of the pre-defined templates (Basic, Ion Boardroom, Minimalist Sales Pitch, Urban Monochrome).
4. **Instant Download** — The finished `.pptx` is saved and made available via a download link.

## Running Locally

### 1. Install dependencies

```bash
pip install -r requirements.txt
pip install fastapi "uvicorn[standard]"
```

### 2. Configure API keys

Create a `.env` file in the project root:

```bash
# Required: API key for the LLM provider used to generate slide content
GOOGLE_API_KEY=your-google-gemini-api-key

# Optional: enables image search for slides
PEXEL_API_KEY=your-pexels-api-key

# Optional: override the default model (default: [gg]gemini-2.5-flash-lite)
SLIDEDECKAI_MODEL=[gg]gemini-2.5-flash-lite
```

### 3. Start the server

```bash
uvicorn api.main:app --reload --port 8123
```

Then open [http://127.0.0.1:8123/](http://127.0.0.1:8123/) in your browser.


## API

### `POST /generate`

Request body:
```json
{
  "topic": "The history of the World Endurance Championship",
  "template_idx": 1
}
```

`template_idx` selects the PPTX template:

| Index | Template               |
|------:|------------------------|
| 0     | Basic                  |
| 1     | Ion Boardroom          |
| 2     | Minimalist Sales Pitch |
| 3     | Urban Monochrome       |

Response:
```json
{
  "file_id": "5cdee493e3a540c28beaf6a3f2fa4f8a",
  "download_url": "/download/5cdee493e3a540c28beaf6a3f2fa4f8a"
}
```

### `GET /download/{file_id}`

Downloads the generated `.pptx` file.

### `GET /health`

Basic health check.


## Notes

- Supported LLMs and provider/API key mappings follow the upstream [SlideDeck AI](https://github.com/barun-saha/slide-deck-ai) project (`src/slidedeckai/global_config.py`).
- Generated files are written to `api/generated/` (not committed to version control).
- This prototype intentionally keeps scope minimal: no authentication, payments, or Canva integration — those belong to other parts of the Tweak platform.
